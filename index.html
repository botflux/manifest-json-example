Génère le manifest depuis https://app-manifest.firebaseapp.com/
  1. Créer le manifest depuis le site
  2. Tu le mets à la racine du projet
  3. Tu ajoute le manifest dans ton index.html (c'est la ligne 4 de ce fichier)
  4. Pour voir si cela a fonctionné regarde dans la console de Chrome. Dans l'onglet Application, il y a un élément "Manifest" dans le menu de gauche. Les informations du manifest seront affiché
